The Evolution of AI Benchmarks and the Emerging Frontier

Benchmarks Evals

Since 2023 GenAI benchmarks have evolved from academic knowledge tests to “work tests” measuring how well a model can do useful paid work. And the next frontier of benchmarking is already emerging - let's take a look!

2023: the knowledge test era

A typical 2023 benchmark consisted of a fixed set of questions with known answers - often multiple choice, making grading the model scores automatic & cheap. For example - MMLU (multiple choice across 57 subjects), HumanEval (small Python coding tasks), GSM8K (grade school math). But these benchmarks essentially only tested stored knowledge (memorization) in simple single turn format - which was fine in 2023 as the models couldn't do much practical work anyway.

2024: the agentic shift

In 2024 models got tool access & learned to complete multi-step tasks. A new generation of benchmarks emerged:

Princeton's SWE-bench (released end of 2023) where the model gets a real GitHub repository with a real bug report and has to produce a fix that passes the project's own tests. Back then the best model solved 1.96% of the tasks.

Sierra's tau-bench made the agent a customer service manager (the agent talked to a simulated user, called APIs + had to follow a written policy).

These benchmarks had their own limitations: the tasks tended to be very small. E.g. fixing 1 small bug. Passing these doesn't help measure whether a model can carry a practical chunk of real work.

2025: long horizon & economic value

Early 2025 benchmarking focus shifted once again this time to “how big of a task can a model handle and is that task worth money”?

This is when METR started measuring model capability in “units of human time” = length of task (measured by how long it takes a person) that a model can complete with a 50% success rate. In early 2025 the frontier stood at about 50 minutes.

The same year OpenAI released GDPval: 1000+ real work tasks across 44 occupations with average human completion time of 7 hours per task.

2026: the new frontier

So what is emerging now? First, we have new benchmarks pushing the difficulty of the frontier. Abundant AI's SWE-Marathon stretches a single task across a roughly 1 billion token codebase. Agents' Last Exam from Berkeley sets the difficulty at real human expert work: 1,000+ tasks following official US occupation categories. Its hardest tier averages a 2.6% full pass rate currently.

However, I see several core shifts beyond that:

Reliability has become a separate eval dimension: Princeton now runs a live Reliability Dashboard on top of its Holistic Agent Leaderboard (HAL), and Sierra's official tau-bench leaderboard now ranks by pass^k.

Evals also started testing raw model and model + harness separately. Terminal-Bench now offers a minimal baseline harness (called Terminus) to separate raw model capability from the harness contribution. HAL measured 34 percent points difference from a single model based on harness alone.

Perhaps most importantly Recursive Self-Improvement is emerging as its own benchmark group. In Ant Group's Meta-Agent Challenge the model gets a sandbox, an evaluation interface + time budget, and has to build an agent that scores well on a hidden test set.
